SolarBoard API & Code

Giới thiệu

Trang web này cung cấp code và API của SolarBoard dành cho tất cả mọi người. Các API có thể được sử dụng để:

  • Tạo board mới
  • Thêm nét vẽ vào board
  • Kiểm tra trạng thái Anti ATS (đã kích hoạt bởi team hay chưa)

Bên cạnh đó, thư viện Python Solar.VTriPO-Panel được cung cấp để quản lý các board mà người dùng tạo ra. VAI (VTriP AI) được thiết kế để quản lý và kiểm soát các thao tác như Subscription, Create Board và nhiều chức năng khác.

API SolarBoard

Tạo Board Mới

POST /api/create-board
Host: api.solarboard.com
Content-Type: application/json

{
  "user_id": "user123",
  "board_name": "New Board"
}

Thêm Nét Vẽ

POST /api/add-drawing
Host: api.solarboard.com
Content-Type: application/json

{
  "board_id": "board123",
  "drawing": {
    "type": "line",
    "start": {"x": 10, "y": 20},
    "end": {"x": 100, "y": 200},
    "color": "#FF0000"
  }
}

Kiểm Tra Trạng Thái Anti ATS

GET /api/check-anti-ats?team=exampleTeam
Host: api.solarboard.com

Hướng dẫn sử dụng Python - Solar.VTriPO-Panel

Dưới đây là ví dụ code sử dụng thư viện Solar.VTriPO-Panel để quản lý board:

import solar_vtripanel as svp

# Khởi tạo đối tượng quản lý board với API key
manager = svp.BoardManager(api_key="your_api_key_here")

# Tạo board mới
new_board = manager.create_board(user_id="user123", board_name="New Board")
print("Board tạo mới:", new_board)

# Thêm nét vẽ vào board
drawing = {
    "type": "circle",
    "center": {"x": 50, "y": 50},
    "radius": 25,
    "color": "#00FF00"
}
result = manager.add_drawing(board_id=new_board["id"], drawing=drawing)
print("Kết quả thêm nét:", result)

# Kiểm tra trạng thái Anti ATS
status = manager.check_anti_ats(team="exampleTeam")
print("Trạng thái Anti ATS:", status)

Lưu ý: Để tắt VAI hoặc bật Anti ATS, người dùng cần mua gói For Studio 270K.

Quy định & Hạn chế

  • Cấm vẽ hình ảnh và nội dung liên quan đến CCN và HỦ kể từ ngày 20/03/2025.
  • VAI (VTriP AI) được sử dụng để quản lý và kiểm soát board, bao gồm Subscription, Create Board và các thao tác liên quan.
  • Hệ thống hỗ trợ các police proxy cơ bản và nâng cao (proxy có thể được cấu hình theo nhu cầu).
