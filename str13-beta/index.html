Mã hóa và Giải mã

Hướng dẫn tích hợp

Để tích hợp phương pháp mã hóa này vào trang web của bạn, hãy thực hiện các bước sau:

  1. Nhúng mã JavaScript của chúng tôi vào trang web của bạn bằng cách sử dụng URL sau:
    2. <script src="https://your-cdn-link/script.js"></script>
  2. Sử dụng đoạn mã sau để mã hóa và giải mã văn bản:
    3. 
<textarea id="inputText"></textarea>
<input type="text" id="key" placeholder="Nhập mã key..."/>
<button onclick="encrypt()">Mã hóa</button>
<button onclick="decrypt()">Giải mã</button>
<textarea id="outputText"></textarea>
